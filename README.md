# Sasogu Apps Landing Page

Landing page estática en HTML5 para la marca **Sasogu Apps**, enfocada en captar clientes para servicios de desarrollo de aplicaciones a medida para pequeños negocios.

## Contenido

- `index.html`: landing page completa en un único archivo
- `favicon.svg`: icono ligero de la marca para navegador y pestañas
- `README.md`: documentación básica del proyecto
- `LICENSE`: licencia del proyecto

## Características

- HTML5 semántico
- CSS y JavaScript integrados en el mismo archivo
- Diseño responsive con enfoque móvil primero
- Navegación con scroll suave
- Formulario conectado a WhatsApp
- Protección básica anti-spam con honeypot y limitación temporal de envíos
- Sin frameworks ni dependencias externas

## Uso local

1. Abre `index.html` directamente en tu navegador.
2. Edita textos, estilos o el número de WhatsApp dentro del mismo archivo si lo necesitas.

## Personalización rápida

### Cambiar número de WhatsApp

Busca esta URL dentro de `index.html`:

```html
https://wa.me/34.........

Y sustitúyela por tu número en formato internacional, sin `+` ni espacios.

### Editar textos de marca

Puedes actualizar fácilmente:

- nombre de marca
- propuesta de valor
- servicios
- testimonios
- textos de llamada a la acción

Todo está centralizado en el contenido de `index.html`.

## Publicación
